### 🎯 How It Works

```mermaid
graph TD
    A[User Query] --> B[Multi-Model Generation]
    B --> C[DeepSeek V3]
    B --> D[Llama 3.2 90B]
    B --> E[Gemini 2.0]
    B --> F[Groq Llama 3.3]
    
    C --> G[Consensus Analysis]
    D --> G
    E --> G
    F --> G
    
    G --> H{Agreement ≥70%?}
    H -->|Yes| I[HIGH Confidence]
    H -->|No| J[LOW Confidence]
    
    I --> K[Synthesize Response]
    K --> L[Verification Check]
    L --> M[Final Answer]
    
    J --> N[Show Uncertainty]
    N --> O[Multiple Perspectives]
```
